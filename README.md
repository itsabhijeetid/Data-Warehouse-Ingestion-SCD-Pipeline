# Data Warehouse Ingestion & SCD Pipeline

**Tools & Technologies:** Microsoft Fabric, Fabric Pipelines, Notebooks, OneLake, Lakehouse, Data Warehouse

- Designed and implemented end-to-end data ingestion pipelines in Microsoft Fabric using Fabric Pipelines and Notebooks.
- Built Slowly Changing Dimension (SCD) Type 1 and Type 2 pipelines with incremental loading strategies for efficient warehouse updates.
- Developed fault-tolerant ingestion workflows to handle partial pipeline failures, ensuring idempotent file processing and preventing duplicate data loads.
- Integrated Lakehouse and Data Warehouse layers within OneLake to support scalable data storage and analytical reporting.
